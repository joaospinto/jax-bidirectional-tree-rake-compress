# Changelog

All notable changes will be documented in this file.

## 0.2.0 — 2026-07-22

- Add selectable rake-only contraction schedules while preserving rake–compress
  as the default.

## 0.1.0 — 2026-07-20

- Add validated CPU preprocessing for deterministic rake–compress plans.
- Add generic PyTree-valued contraction and balanced sibling reduction.
- Add residual tapes and reverse expansion for bidirectional recovery.
- Support composition with JAX JIT, vectorization, and differentiation.
- Add subtree-sum and affine-broadcast examples, tests, and benchmarks.
- Add typed package metadata, cross-platform wheel validation, and PyPI Trusted
  Publishing through GitHub Releases.
